> Typescript Based Slot Game

## Built with

- Node v10.16.0
- Yarn 1.19.2
- Npm 6.9.0
- Typescript
- Spine
- Pixi.js
- XState
- Babel
- Webpack

## Install

```sh
$ yarn
```

## Build Development

```sh
$ yarn build
```

or

```sh
$ yarn watch
```

## Usage

```sh
$ yarn serve
```

**Open http://localhost:8080 in browser and click on dist/ directory**
